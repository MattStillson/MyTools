How to monitor system performance and write to SQL Server using Powershell (PowerShell)

Introduction

This script demonstrates how to monitor system performance and write to SQL Server using Powershell. The script lets you quickly identify hardware bottlenecks.

It contains Processor, Memory, Disk, Number of transactions and Batch requests and collects every 30 minutes for 365 days.

Scenarios

As some people want to know how to monitor system performance and write to SQL Server using Powershell, this script will provide some help for them.

Script

Step 1: Open CreateScheduleJob.ps1 and then modify MinitorSystemPerformance.ps1 path.

Step 2: Run Windows PowerShell as administrator.

Step 3: Then input the path of CreateScheduleJob.ps1 and press Enter.

 

Here are some code snippets for your reference. 

- PowerShell code snippet -

$ServerName = "WS-HAMMEZH-02"

$SQLDBName = "test"

$UserID = "sa"

$Password = "88888888"

$SqlConnection = New-Object System.Data.SqlClient.SqlConnection

$SqlConnection.ConnectionString = "Server = $ServerName; Database = $SQLDBName; User ID= $UserID; Password= $Password"

$SqlCmd = New-Object System.Data.SqlClient.SqlCommand

$SqlCmd.CommandText = "

IF NOT EXISTS(SELECT 1 FROM sys.objects WHERE object_id = OBJECT_ID('dbo.SystemPerforman'))

BEGIN

       CREATE TABLE [dbo].[SystemPerforman]

       (

              [Id] INT Identity NOT NULL PRIMARY KEY,

              [Timestamp] DATETIME NOT NULL,

              [ServerName] NVARCHAR(100) NOT NULL,

              [ProcessorTime(%)] DECIMAL(8, 2) NOT NULL,

              [ProcessorPTime(%)] DECIMAL(8, 2) NOT NULL,

              [ProcessorUTime(%)] DECIMAL(8, 2) NOT NULL,

              [ProcessorQueueLength] INT NOT NULL,

              [AvailableMBytes(MB)] INT NOT NULL,

              [PageFaults] INT NOT NULL,

              [DiskIdleTime(%)] DECIMAL(8, 2) NOT NULL,

              [DiskReadsSec] INT NOT NULL,

              [DiskWritesSec] INT NOT NULL,

              [AvgDiskReadQuenLength] INT NOT NULL,

              [AvgDiskWriteQuenLength] INT NOT NULL,

              [Transactions] INT NULL,

              [LockAvgWaitTime(ms)] INT NULL,

              [BatchRequests] INT NULL,

              [PhysicalReadsSec] INT NULL

       )

END "

$SqlCmd.Connection = $SqlConnection

$SqlAdapter = New-Object System.Data.SqlClient.SqlDataAdapter

$SqlAdapter.SelectCommand = $SqlCmd

$DataSet = New-Object System.Data.DataSet

$SqlAdapter.Fill($DataSet) | Out-Null

 

- end -
